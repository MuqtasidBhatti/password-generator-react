# 🔐 Random Password Generator (React + Vite)

![React](https://img.shields.io/badge/React-18-blue)
![Vite](https://img.shields.io/badge/Vite-Build%20Tool-purple)
![License](https://img.shields.io/badge/License-MIT-green)

A modern and responsive Random Password Generator built using **React** and **Vite**.  
It generates secure passwords based on selectable difficulty levels and demonstrates the use of React Hooks, conditional rendering, and dynamic state management.

---

## 🚀 Live Demo

👉 https://password-generator-react-smoky.vercel.app/

---

## ✨ Features

- 🔹 Easy Mode → 5 character random password  
- 🔹 Medium Mode → 7 character password including numbers  
- 🔹 Hard Mode → 12 character secure password including numbers  
- 🔹 Random Mode → Automatically selects between Easy / Medium / Hard  
- 🔹 Copy to Clipboard functionality  
- 🔹 Error handling (if no option selected)  
- 🔹 Clean gradient UI design

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/MuqtasidBhatti/password-generator-react.git
cd password-generator-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## 🎯 How It Works

1. Select a difficulty mode  
2. Click **Generate Password**  
3. Click **Copy** to copy the generated password
